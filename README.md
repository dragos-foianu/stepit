##### Repository Local

###### Initializare
git init

###### Vedem modificari
git status

###### Istorie
git log

###### Diferente
git diff

###### Adaugare fisier in sistem
git add <file>
git add .

###### Commit
git commit -m "mesaj"

##### Repository Remote

###### Repository Remote (Github)

###### Clonare
git clone https://github.com/dragos-foianu/stepit.git

###### Trimitere modificari pe server
git push

###### Actualizare modificari locale (iau de pe server)
git fetch
git pull
